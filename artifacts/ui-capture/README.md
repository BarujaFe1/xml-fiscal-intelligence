# UI Capture

Before/after evidence for the automated fiscal reconciliation UX capture.

## `before/` — user-provided

The `before/` package is the **user-provided** `ui-capture.zip`. It is **not**
part of this repository and must be placed here manually when available:

```
artifacts/ui-capture/before/   # unzip ui-capture.zip here
```

Do **not** fabricate before/ images. This folder is intentionally left empty
until the user supplies `ui-capture.zip`.

## `after/` — generated

The `after/` directory is generated by `npm run screenshots:all` (see
`artifacts/ui-capture/after/README.md` for the full structure and commands).

## Workflow

1. Obtain `ui-capture.zip` from the user and unzip into `artifacts/ui-capture/before/`.
2. Start the app (`npm run start`) and optionally seed data (`npm run seed:demo`).
3. Run `npm run screenshots:all` to produce `after/`.
4. Compare `before/` vs `after/` (gallery + contact sheet) for UX evidence.
